Django를 사용해 게시판, 회원가입 및 로그인, 유저페이지를 구현했습니다.
Django 기본 기능에 대한 복습 차원에서 만든 페이지입니다.
부트캠프에서 Django를 사용한 프로젝트를 많이 경험했지만,
혼자서 기능 하나하나에 대해 이해해보고 싶었습니다.

서버 개발 과정에서 웹 프레임워크의 유용함을 체감할 수 있었습니다.
특히 내장된 유저 인증 시스템은 캐시의 사용이나 암호화와 같은 과정을 직접 구현하지 않고도 유저관리를 할 수 있게 했습니다.

text project

일경험 기관의 1년 간의 민원추세를 파악하기 위한 프로젝트입니다.
엑셀 파일형식의 원본 파일을 분석하고 시각화하면서 키워드를 추출하고(Extract), 기간에 따라 분류해(Transform), 시각화 도구에서 사용할 수 있도록 저장(Load)하는 데이터 ETL의 과정을 경험했습니다.
또한, 프로젝트 기획부터 코드 작성 및 결과물 산출까지 모든 과정을 혼자서 진행한 프로젝트였습니다. 특히 기초부터 결론까지 코드를 작성하는 건 처음이었기 때문에 막막했지만, 결과적으로 스스로 생각해서 코드를 작성하는 능력을 가지게 해준 경험이었습니다.

site project

협업을 통해 Django를 백엔드로 사용하는 서비스를 개발하는 프로젝트였습니다. 원두판매서비스를 만들었으며, 상품정보는 이미 존재하는 사이트에서 크롤링해서 사용했습니다.
상품 정보를 크롤링하여 DB에 저장하기 위해 간단한 데이터 파이프라인을 만들었습니다. 크롤링한 요소를 적절하게 반환하는 함수를 작성하고, 반환된 요소를 옮길 수 있는 DB를 설계했습니다. 적절한 DB 구조를 설계하는게 많이 중요하다고 느꼈고, DB 설계에 대한 경험을 쌓을 수 있는 기회였습니다.. 그 밖에도 기본적인 CRUD를 사용해 장바구니 기능도 구현했습니다.
